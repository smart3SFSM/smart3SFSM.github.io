private

The number of iterations the algorithm does at each pyramid level. (If the flag 'useSmartIterationDistribution' is set, the number of iterations will be redistributed in such a way, that more iterations will be done on more coarser levels.)

int VARAdvancedDialog::penalization
private

The useed penalization (or regularization) method. Possible values: PENALIZATION_TICHONOV - linear smoothness; PENALIZATION_CHARBONNIER - non-linear edge preserving smoothness; PENALIZATION_PERONA_MALIK - non-linear edge-enhancing smoothness. (This parameter is ignored if flag 'useAutoParams' is set).

float VARAdvancedDialog::phi
private

The smoothness parameter, or the weight coefficient for the smoothness term.

int VARAdvancedDialog::pixelWindowSize
private

Size of the pixel neighbourhood used to find polynomial expansion in each pixel. The larger values mean that the image will be approximated with smoother surfaces, yielding more robust algorithm and more blurred motion field. Typically, pixelWindowSize = 3, 5 or 7.

int VARAdvancedDialog::pyramidLevels
private

The number of pyramid layers, including the initial image. pyramidLevels=1 means that no extra layers are created and only the original images are used. This parameter is ignored if flag 'useAutoParams' is set.

double VARAdvancedDialog::pyramidScale
private

Specifies the image scale (<1) to build the pyramids for each image. pyramidScale=0.5 means the classical pyramid, where each next layer is twice smaller than the previous. (This parameter is ignored if flag 'useAutoParams' is set).

double VARAdvancedDialog::standardDeviation
private

Standard deviation of the Gaussian that is used to smooth derivatives that are used as a basis for the polynomial expansion. For pixelWindowSize=5 you can set standardDeviation=1.1 , for pixelWindowSize=7 a good value would be standardDeviation=1.5.
