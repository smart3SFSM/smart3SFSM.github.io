Default scaling parameter. If it is -1 or absent, the function performs the default scaling. Otherwise, the parameter should be between 0 and 1. INIT_RECTIFY_ALPHA=0 means that the rectified images are zoomed and shifted so that only valid pixels are visible (no black areas after rectification).
