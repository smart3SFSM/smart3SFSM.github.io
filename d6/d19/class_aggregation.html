Detailed Description

The Aggregation class computes the cost aggregation for the AD-Census stereo matching algorithm.

Parameters
leftImageLeft stereo image.
rightImageRight stereo image.
colorThreshold1Preset threshold value for the color difference between the target pixel p and the referece pixel p1 on the cross arms used by the cross-based cost aggregation to compute the shape of the pixel's support region. colorThreshold1 ist used as a large value to include enough pixels for textureless regions (colorThreshold1 > colorThreshold2).
colorThreshold2Preset threshold value for the color difference between the target pixel p and the referece pixel p1 on the cross arms used by the cross-based cost aggregation to compute the shape of the pixel's support region. colorThreshold2 ist used as the default value for well textured regions (colorThreshold2 < colorThreshold1).
maxLength1Preset maximum length (measured in pixels) between the target pixel p and the referece pixel p1 on the cross arms used by cross-based cost aggregation to compute the shape of the pixel's support region. maxLength1 ist used as a large value to include enough pixels for textureless regions (maxLength1 > maxLength2).
maxLength2Preset maximum length (measured in pixels) between the target pixel p and the referece pixel p1 on the cross arms used by cross-based cost aggregation to compute the shape of the pixel's support region. maxLength2 ist used as the default value for well textured regions (maxLength2 < maxLength1).

int Aggregation::colorThreshold1
private

Preset threshold value for the color difference between the target pixel p and the referece pixel p1 on the cross arms used by the cross-based cost aggregation to compute the shape of the pixel's support region. colorThreshold1 ist used as a large value to include enough pixels for textureless regions (colorThreshold1 > colorThreshold2).
