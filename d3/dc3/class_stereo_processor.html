aggregatingIterationsNumber of iterations for the cost aggregation.
colorThreshold1Preset threshold value for the color difference between the target pixel p and the referece pixel p1 on the cross arms used by the cross-based cost aggregation to compute the shape of the pixel's support region. colorThreshold1 ist used as a large value to include enough pixels for textureless regions (colorThreshold1 > colorThreshold2).
colorThreshold2Preset threshold value for the color difference between the target pixel p and the referece pixel p1 on the cross arms used by the cross-based cost aggregation to compute the shape of the pixel's support region. colorThreshold2 ist used as the default value for well textured regions (colorThreshold2 < colorThreshold1).
maxLength1Preset maximum length (measured in pixels) between the target pixel p and the referece pixel p1 on the cross arms used by cross-based cost aggregation to compute the shape of the pixel's support region. maxLength1 ist used as a large value to include enough pixels for textureless regions (maxLength1 > maxLength2).
maxLength2Preset maximum length (measured in pixels) between the target pixel p and the referece pixel p1 on the cross arms used by cross-based cost aggregation to compute the shape of the pixel's support region. maxLength2 ist used as the default value for well textured regions (maxLength2 < maxLength1).
colorDifferenceThreshold value for the color difference between the corresponding pixels in the original images used by the scanline optimization.
pi1Constant value that is used as a computation parameter for penalizing the disparity changes between neighboring pixels in the scanline optimization. It is used in several ways depending on the comparison between the computed color difference and the 'colorDifferene'-threshold.
pi2Constant value that is used as a computation parameter for penalizing the disparity changes between neighboring pixels in the scanline optimization. It is used in several ways depending on the comparison between the computed color difference and the 'colorDifferene'-threshold.
dispToleranceTolerance value for the computed disparities of the left and the right image used by the outlier detection.
votingThresholdThreshold for the minimum number of reliable pixels in the support region used by the iterative region voting.
votingRatioThresholdThreshold for the minimum ratio between the most frequent disparity value and the number of reliable pixels in the support region used by the iterative region voting.
maxSearchDepthMaximum search depth parameter used by the proper interpolation to detect the remaining outliers.
blurKernelSizeBlurring kernel size used by the OpenCV blur funtion for the depth discontinuity adjustment.
cannyThreshold1Threshold value used by the canny edge detector (OpenCV implementation) for the depth discontinuity adjustment.
cannyThreshold2Threshold value used by the canny edge detector (OpenCV implementation) for the depth discontinuity adjustment.
cannyApertureSizeCanny aperture size used by the canny edge detector (OpenCV implementation) for the depth discontinuity adjustment.
