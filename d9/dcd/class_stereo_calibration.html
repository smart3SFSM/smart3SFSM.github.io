const double StereoCalibration::INIT_RECTIFY_ALPHA = 1
staticprivate

Default scaling parameter. If it is -1 or absent, the function performs the default scaling. Otherwise, the parameter should be between 0 and 1. INIT_RECTIFY_ALPHA=0 means that the rectified images are zoomed and shifted so that only valid pixels are visible (no black areas after rectification). INIT_RECTIFY_ALPHA=1 means that the rectified image is decimated and shifted so that all the pixels from the original images from the cameras are retained in the rectified images (no source image pixels are lost). Obviously, any intermediate value yields an intermediate result between those two extreme cases.

Default zero zone for the cv::cornerSubPix method. The zero zone is half of the size of the dead region in the middle of the search zone over which the summation in the formula below is not done.
